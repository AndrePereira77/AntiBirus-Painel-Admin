# 🖥️ Gestão de Inventário — Loja de Informática

Projeto de aplicação desktop para gerir o inventário e as vendas de uma loja de informática.

O tema do projeto é o controlo de stock de produtos de informática, com atualização rápida do stock através de ações de venda e reposição.

## Tecnologias utilizadas

* **Python 3.10+**
* **PyQt5** para interface gráfica
* **MySQL** para armazenamento de dados
* **mysql-connector-python** para ligação ao banco de dados
* **UI em XML** para construção das janelas (login, splash, dashboard)

## Assunto

Este projeto foca-se na gestão de um pequeno comércio de informática, incluindo:

* inventário de produtos
* controlo de stock
* atualizações de stock ao vender ou repor
* interface desktop simples para operação diária
